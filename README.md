# Antonio Vazquez Engineering Portfolio

Personal engineering portfolio for Antonio Vazquez, a Texas A&M Mechanical Engineering student focused on medical device R&D, mechanical design, catheter systems, surgical robotics, testing, and prototyping.

## Local Preview

Open `index.html` in a browser, or run a simple static server from the project root.

## Build Check

```bash
npm run build
```

The build script validates local HTML links, assets, and basic page metadata. It does not require external packages.

## Deploying on Vercel

Import this repository into Vercel as a static site.

- Framework preset: Other
- Build command: `npm run build`
- Output directory: leave empty

## Editing Notes

- Main homepage copy lives in `index.html`.
- Web resume content lives in `resume.html`.
- PDF resume lives at `assets/Antonio_Vazquez_Resume.pdf`.
- Case studies live in `projects/`.
- Replace visual placeholders with CAD renders, prototype photos, test plots, diagrams, or annotated screenshots when ready.
